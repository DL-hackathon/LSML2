1. Project description

In the final project we will implement a model to classify whether a text, typed by user, is hate speech text or non-hate.

It seems to be an actual task nowadays since people communicate a lot in social media and kiberbulling spreads widely.

2. Multi-lingual HateSpeech Dataset

To learn our model we will use Hate Speech Dataset in Multiple Languages.

Dataset contains hate speech text with labels, where 0 represents non-hate and 1 shows hate texts.
The following are the languages in the dataset:
Arabic, English, Chinese, French, German, Russian, Turkish, Roman Hindi/Urdu, Korean, Italian, Spanish, Portuguese, Indonesian - 13 in total.
Number of samples:
- for training: 219 981;
- for testing:   24 449.

3. Model description

As a base model we will use "BERT multilingual base model (uncased)": pretrained model on the top 102 languages with the largest Wikipedia.

For a base line we will use BERT produced embeddings as features for CatBoostClassifier and classify whether text is hate or not.
For a more advanced approach, we will fine-tune the base model on our dataset to adapt the model to our downstream task. In this case, we will use BERT as a classifier, modifying its top layer for binary classification purposes.
